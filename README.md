## 线路类型说明

### 普通中转线路
- 以中转节点为主
- 成本相对更低、流量选择更充足
- 设备数限制更明确
- 适合日常浏览、搜索、轻度内容访问等场景

### 专线线路
- 独立专线或优先调度路由
- 高峰期稳定性通常更好
- 峰值速率与设备数量限制更少（以官方说明为准）
- 更适合对稳定性要求更高的场景（例如多设备、长时间使用）
