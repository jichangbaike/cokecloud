## 线路类型说明

### 普通中转线路
- 以中转节点为主
- 通常成本更低、流量选择更充足
- 设备数限制更明确
- 适合日常浏览、搜索、轻度使用等场景

### 专线线路
- 独立专线或优先调度路由（以官方说明为准）
- 高峰期稳定性通常更好
- 峰值速率与设备数量限制更少（以官方说明为准）
- 更适合对稳定性要求更高的场景（例如多设备、长时间使用）

### 中转 vs 专线对比

| 维度 | 普通中转线路 | 专线线路 |
|---|---|---|
| 定位 | 性价比 / 流量优先 | 稳定性 / 体验优先 |
| 高峰期表现 | 视负载波动 | 通常更稳（视地区/负载而定） |
| 速率/设备限制 | 常见限制更明确 | 常见限制更少（以官方为准） |
| 适合人群 | 入门、预算优先、轻度使用 | 多设备、长时间使用、体验敏感 |

> 说明：线路体验会受地区、时段、网络环境与节点负载影响。本文仅作信息整理与对比参考。

## 常见问题（FAQ）

### 1）普通中转和专线，核心区别是什么？
一般来说，普通中转更偏“性价比/流量优先”，专线更偏“稳定性/体验优先”。体验差异会受地区、时段与负载影响，建议先短周期测试。

### 2）设备数“5 台”是怎么计算的？
通常以同时在线或绑定/登录设备数为口径（不同服务商规则可能不同）。请以官方规则为准；多设备用户更建议关注设备限制条款。

### 3）峰值速率 200Mbps/300Mbps 在实际使用中意味着什么？
峰值速率是上限参考值。实际速度会受本地网络、运营商、距离、节点负载等影响，建议用常用网络环境测试后再决定周期。

### 4）为什么会出现速度波动？
常见原因包括高峰时段拥塞、节点负载变化、本地网络波动、跨地区链路质量变化等。对稳定性敏感可优先考虑体验导向的线路方案。

### 5）月付/季付/半年付/年付怎么选？
不确定体验时先月付测试；确认满足需求后再升级长周期以降低平均成本。若你更看重稳定性，优先把线路类型选对，再决定周期。

### 6）月流量 100GB/200GB/300GB/500GB 怎么判断够不够？
网页浏览与轻度使用相对省流量；高清视频/大文件下载会显著增加用量。建议以你的真实月用量为准，选择“略有富余”的档位更省心。

### 7）“流量重置”是什么意思？
通常表示当月流量用尽后，可按官方规则额外购买/恢复一定流量。具体价格与规则以官方说明为准。若你经常需要重置，通常说明套餐档位偏小。

### 8）节点地区会固定不变吗？
节点覆盖通常会随服务策略调整。本仓库中的地区信息为整理时参考，请以官方实际为准。

### 9）信息如果与官方不一致，以谁为准？
以官方为准。本仓库仅做信息整理与对比参考，可能存在滞后或更新不及时。
